== Description ==
<a href="http://www.horoscop.ro/">Horoscop</a> pe site la tine prin <strong>Horoscopul Zilnic</strong> in limba Romana pe site-ul tau prin instalarea acestui plugin de wordpress care preia automat feedul de pe horoscop.ro.

Instalarea dureaza doar cateva minute iar aspectul poate fi customizat conform aspectului site-ului tau.

== Installation ==
1. Descarca 'horoscopul-zilnic-ro.zip' si salveaza-l local
2. Navigheaza in directoarele website-ului prin FTP apoi incarca 'horoscopul-zilnic-ro.zip' in directorul '/wp-content/plugins/'
3. Activeaza pluginul 'Horoscopul Zilnic RO' din sectiunea 'Plugins' sau 'Module' in WordPress
4. Configureaza pluginul in pagina special creata din 'Settings' sau 'Setari' pentru o afisare cat mai potrivita temei
5. Utilizarea pluginului se poate face in 2 moduri: (1) prin utilizarea Widget-ului "Horoscopul Zilnic" din sectiunea 'Aspect'/'Widget-uri' sau 'Appearance'/'Widgets' (2) prin folosirea shortcode-ului [horoscop-zilnic] utilizand setarile din pagina pluginului sau [horoscop-zilnic arg1=value arg2=value ...etc] folosind argumentele pentru a modifica setarile pluginului direct in pagina. Posibilele argumente sunt vizibile in pagina pluginului

== Frequently asked questions ==
Vrei <a href="http://www.horoscop.ro/">Horoscop Zilnic</a> in romana afisat pe site? Este foarte simplu de utilizat. Urmeaza instrunctiunile de la instalare si cu siguranta in doar cateva minute poti avea zodia ta preferata de la horoscop.ro, actualizat zilnic.
